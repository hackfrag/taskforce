This ist a pre alpha. It is heavily inspired by all the GTD apps on a mac. Like Things, OmniFocus and Hitlist.

Currently only support for Firefox 3+ / Safari 3.2+ / Google Chrome

Google Gears is used for database storage.

Quick Shortcuts:
  • Alt+w : New task
  • Alt+e : Edit the current task
  • Enter : Save the current task
  • Esc : Cancel the edit mode
  • Space : Set the current todo done/undone
  • Arrows Keys up/down for navigation
  • Delete: Delete the current task
  • Contextmenu for other functions
