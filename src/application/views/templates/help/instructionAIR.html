This ist a pre alpha. It is heavily inspired by all the GTD apps on a mac. Like Things, OmniFocus and Hitlist.

Thanks to jQuery,jQuery-ui,FilamentGroup for the nice slider, Panic for Coda & Transmit and all the GTD apps Things, OmniFocus, Hitlist.

Quick Shortcuts:
  • Ctrl+n : New todo
  • Ctrl+e : Edit the current todo
  • Enter : Save the current todo
  • Esc : Cancel the edit mode
  • Space : Set the current todo done/undone
  • Arrows Keys up/down for navigation
  • Delete: Delete the current todo
